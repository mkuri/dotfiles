# Sub-agent Model Policy

When dispatching sub-agents using the Task tool for plan execution
(subagent-driven-development, executing-plans, dispatching-parallel-agents):

- **Implementer subagents**: Use `model: "sonnet"` to reduce costs
- **Spec reviewer subagents**: Use `model: "opus"` for accurate review
- **Code quality reviewer subagents**: Use `model: "opus"` for accurate review
